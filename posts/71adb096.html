函数

北川2025-04-292025-10-31
  1. 函数的完整语法

在 Java 中，函数也被称为方法（method），其完整语法如下：

  • 访问修饰符（可选）：控制方法的访问权限，常见的有public（公共的，任何类都可以访问）、private（私有的，只有在本类内部可以访问）、protected（受保护的，在本类、同包类以及子类中可以访问）以及默认（不写修饰符，同包类可以访问）。
  • 修饰符（可选）：例如static（表示该方法是静态方法，可以通过类名直接调用，而不需要创建类的实例）、final（表示该方法不能被子类重写）等。
  • 返回类型：指定方法执行后返回的数据类型，如果方法不返回任何值，则使用void关键字。
  • 方法名：遵循 Java 的命名规范，通常采用驼峰命名法，首字母小写，后续单词首字母大写。
  • 参数列表（可选）：由零个或多个参数组成，每个参数由参数类型和参数名组成，多个参数之间用逗号分隔。例如：(int num1, String str)。
  • throws 异常列表（可选）：用于声明该方法可能抛出的异常。
  • 方法体：包含实现方法功能的语句块。
  • return 语句（如果返回类型不是void，则必须有）：用于返回方法的执行结果，返回值的数据类型必须与返回类型一致。
  1. 形参和实参的区别
  • 形参（形式参数）：在方法声明中定义的参数，用于接收调用方法时传入的值。形参只在方法内部可见，其作用域仅限于该方法。例如在方法public static int add(int a, int b)中，a和b就是形参，它们代表了在方法调用时会传入的具体数值。
  • 实参（实际参数）：在调用方法时传递给方法的实际值。实参可以是常量、变量、表达式等，其数据类型必须与形参的类型兼容。例如在int result = add(3, 5);中，3和5就是实参，它们是实际参与方法运算的具体数值。
  1. 参数列表的组成部分

参数列表由零个或多个参数组成，每个参数由以下两部分组成：

  • 参数类型：指定参数的数据类型，可以是基本数据类型（如int、double、boolean等）或引用数据类型（如String、自定义类等）。
  • 参数名：遵循 Java 的命名规范，用于在方法体中引用该参数。例如，在public void printMessage(String message)中，String是参数类型，message是参数名。

多个参数之间用逗号分隔，如

public static int multiply(int num1, int num2)。

  1. 返回值的使用
  • 返回值类型匹配：方法的返回值类型必须与方法声明中的返回类型一致。如果返回类型是void，则方法中不能有return语句返回具体值，但可以有return;语句用于提前结束方法执行。例如：

public static double calculateArea(double radius) { return 3.14 * radius * radius; }

这里方法calculateArea返回一个double类型的值，与方法声明的返回类型一致。

  • 接收返回值：在调用有返回值的方法时，通常需要一个变量来接收返回值，以便后续使用。例如：

double area = calculateArea(5.0); System.out.println(“圆的面积是: “ + area);

  1. 参数和返回值的使用时机
  • 参数：当方法功能执行中有不确定的数据时，该数据应该提炼为参数。这样可以使方法更加通用，能够处理不同的数据输入。例如，上述的add方法可以接收不同的整数进行加法运算，而不是固定计算某两个数的和。如果方法内部需要处理不同的文本信息，那么可以将文本信息作为参数传入，如public void printMessage(String message)方法可以打印不同的消息内容。
  • 返回值：当调用者需要使用方法的执行结果时，需要将其返回。例如，在计算圆面积的方法calculateArea中，调用者可能需要获取计算出的面积值用于后续的计算或显示，因此方法返回计算结果。如果方法只是执行一些操作，如打印日志、更新数据库等，不需要向调用者返回特定的结果，那么可以将返回类型定义为void。
